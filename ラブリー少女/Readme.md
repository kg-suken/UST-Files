# ラブリー少女
ラブリー少女のファイルです。
tssln(Voisona)のファイルがオリジナルなので基本的にこれを使用することをおすすめします。
うまく発音できない場合などは、連続音なしのustファイルを利用してください。

ファイルが対応していない場合は[UtaFormatix](https://sdercolin.github.io/utaformatix3/)にて変換を行うことができます。
